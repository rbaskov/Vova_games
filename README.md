# Хроники Эльдории

Пиксельная Action-RPG в браузере. HTML5 Canvas + Vanilla JavaScript. Без зависимостей.

## Запуск

```bash
# Локальный сервер (нужен для ES modules)
python3 -m http.server 8080
# или
npx serve .
```

Открыть `http://localhost:8080` в браузере.

### Docker (Synology / сервер)

```bash
docker-compose up -d --build
# Доступно на порту 3040
```

## Управление

| Клавиша | Действие |
|---------|----------|
| WASD / Стрелки | Движение |
| Пробел | Атака |
| E | Говорить с NPC |
| Q | Использовать зелье |
| 1, 2, 3 | Способности стихий |
| I | Инвентарь |
| J | Журнал квестов |
| H | Справка |
| M | Музыка вкл/выкл |
| ESC | Выход в меню (с сохранением) |

На мобильных устройствах -- боковые панели с виртуальным джойстиком и кнопками. Игровая область сохраняет пропорции 4:3.

## Контент

### Локации (9)
- **Деревня Брайтхолл** -- стартовая зона, NPC, магазины
- **Королевский замок** -- мирная зона, Король, наёмники, конюх, торговцы
- **Темный лес** -- слаймы, волки, бандиты, босс: Лесной страж
- **Огненное ущелье** -- скелеты, бандиты, босс: Огненный дракон
- **Ледяная пещера** -- големы, бандиты, босс: Ледяной лич
- **Замок Темного мага** -- стражники, рыцарские стражи, босс: Темный маг
- **Адская яма** -- босс: ЗлойРокДемон (10M HP)
- **Арена** -- бесконечные раунды с гладиаторами, 100$ за победу
- **Подземелья** -- процедурная генерация, бесконечные уровни

### Оружие (18+ видов)
- **Мечи:** Железный, Стальной, Мифриловый, Костяной (дроп), Ржавый бандита (дроп), Меч РокДемона (дроп)
- **Копья:** Обычное, Огненное, Рыцарское, Грубое бандита (дроп)
- **Луки:** Лук, Арбалет, Потрепанный бандита (дроп)
- **Топоры:** Железный, Стальной, Гладиатора, Рыцарский, Ржавый бандита (дроп)
- **Двуручники:** Железный, Стальной, Рыцарский, Меч Бальдионидов (дроп с Лича)

### Доспехи (7 тиров)
- Кожа -> Кольчуга -> Железо -> Гладиатор -> Мифрил -> Рыцарские -> **Доспехи богов**
- Шлем Ферида (+800 DEF), Кольчуга Эзаниллы (+1000 DEF), Поножи Иомерида (+900 DEF)

### Щиты (7 видов, каждый с уникальным визуалом)
- Деревянный (20%) -> Железный (40%) -> Огненный (70%) -> Мифриловый (60%) -> Зеркальный (80%, отражает) -> Щит Моремирида (100%, x3) -> Щит РокДемона (60%, +2000 DEF, дроп)

### Враги (12+ типов)
- Слайм, Волк, Скелет (с мечом), Голем
- Бандит-мечник, Бандит-копейщик, Бандит-лучник, Бандит-топорщик
- Рыцарский стражник (копьё + щит, 200 HP)
- Гладиатор (арена)
- 6 уникальных боссов с фазами и диалогами перед боем

### Наёмники (8 видов)
Нанимаются в Королевском замке за 100 монет. Макс. количество = уровень + 1.

| Наёмник | Оружие | HP | Особенность |
|---------|--------|----|-------------|
| Дарен | Меч | 120 | Сбалансированный |
| Рольф | Копьё | 100 | Дальний ближний бой |
| Ивар | Лук | 80 | Стреляет издалека |
| Гром | Топор | 150 | Сильнейший удар |
| Бронк | Щит | 250 | Танк |
| Зефир | Меч | 70 | Сверхбыстрый |
| Лиана | Посох | 60 | Лечит +15 HP игроку, +10 отряду |
| Аркан | Магия | 65 | Дальние магические атаки |

Наёмники смертны -- у них есть HP, враги и боссы их атакуют.

### Системы
- Выбор класса при старте игры
- Прокачка (10 уровней, HP/ATK растут)
- 3 способности стихий (Щит Земли, Огненный шар, Ледяная волна)
- Инвентарь с экипировкой и торговлей
- 10 мини-квестов с трекером
- Чекпоинты (респавн при смерти)
- Сохранение в localStorage
- Процедурные подземелья (бесконечная глубина)
- Арена с волновой системой
- Боевой конь (скорость + урон, покупка за 300$)
- Наёмники-компаньоны (следуют, атакуют, умирают)
- Диалоги с боссами перед битвой
- Звуки и музыка (Web Audio API, без файлов)
- Режим Песочница (бессмертие, все предметы)

## Мобильная версия

- Боковые панели: джойстик слева, кнопки справа
- Игровая область сохраняет пропорции 4:3 (не растягивается)
- Кнопка меню (☰) для выхода с сохранением
- Подсказка повернуть экран в портретном режиме

## Технологии

- HTML5 Canvas
- Vanilla JavaScript (ES6 modules)
- Web Audio API (процедурные звуки)
- Шрифт Press Start 2P (Google Fonts)
- Docker + nginx:alpine для деплоя
- Ноль внешних зависимостей
