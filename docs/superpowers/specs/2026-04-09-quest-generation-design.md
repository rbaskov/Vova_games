# Генерация квестов — Спецификация

**Статус:** Планируется (после рандомных событий)

## Цель

Добавить бесконечные рандомно-генерируемые квесты поверх существующих 10 ручных. NPC выдают новые задания после выполнения предыдущих. Сложность масштабируется по уровню игрока.

## Архитектура

Новый файл `js/questgen.js`. Генерирует квест-объект из шаблона + рандомных параметров. Совместим с существующей системой квестов в `js/quests.js`.

```
NPC (Старейшина/Король) → "Есть задание!" → генерируем квест из шаблона
                                            → добавляем в player.quests
                                            → отображаем в трекере
```

## Типы квестов

### 7 шаблонов

| Тип | Шаблон текста | Цель | Пример |
|-----|--------------|------|--------|
| Охота | "Убей {N} {враг} в {локация}" | Убить N врагов определённого типа | "Убей 5 волков в Тёмном лесу" |
| Сбор | "Собери {N} {предмет}" | Убить врагов, собрать дроп | "Собери 3 клыка волка" |
| Доставка | "Отнеси {предмет} от {NPC_A} к {NPC_B}" | Поговорить с NPC_B | "Отнеси письмо Королю" |
| Зачистка | "Зачисти {локация}" | Убить всех врагов на карте | "Уничтожь всех бандитов в ущелье" |
| Арена | "Продержись {N} раундов" | Дожить до раунда N | "Пройди 10 раундов на арене" |
| Эскорт | "Проведи {NPC} до {локация}" | NPC идёт за игроком, довести живым | "Доведи торговца до деревни" |
| Босс-рематч | "Победи {босс} снова" | Босс респавнится, убить заново | "Победи Лесного стража ещё раз" |

### Генерация параметров

Параметры зависят от уровня игрока:

| Параметр | Уровень 1-3 | Уровень 4-6 | Уровень 7-10 |
|----------|-------------|-------------|--------------|
| N (кол-во) | 3-5 | 5-10 | 8-15 |
| Враги | Слаймы, волки | Скелеты, бандиты | Големы, элитные |
| Локации | Лес | Ущелье, пещера | Замок, подземелья |
| Раунды арены | 5 | 10 | 20 |

## Награды

| Сложность | Монеты | XP | Доп. награда |
|-----------|--------|----|----|
| Лёгкий (уровень 1-3) | 20-50 | 15-30 | 1-2 зелья |
| Средний (уровень 4-6) | 50-150 | 30-80 | Рандомное оружие текущего тира |
| Сложный (уровень 7-10) | 150-500 | 80-200 | Редкая броня или эликсир |

## Генерация названий

Комбинация прилагательного + существительного:

```
Прилагательные: Опасная, Срочная, Тайная, Кровавая, Дерзкая, Отчаянная
Существительные: охота, вылазка, миссия, зачистка, разведка, доставка
```

Пример: "Кровавая охота" — "Убей 8 скелетов в Ледяной пещере"

## NPC-квестодатели

| NPC | Локация | Типы квестов | Сколько одновременно |
|-----|---------|-------------|---------------------|
| Старейшина | Деревня | Охота, сбор, доставка | 1 активный |
| Король Альдрик | Замок | Зачистка, босс-рематч, эскорт | 1 активный |
| Страж подземелий | Деревня | Арена, зачистка подземелья | 1 активный |

После выполнения квеста — NPC сразу предлагает новый (другого типа).

## Отличие от ручных квестов

| | Ручные | Генерируемые |
|---|--------|-------------|
| Количество | 10, фиксированные | Бесконечные |
| Цвет в трекере | Золотой | Серебряный |
| Сюжет | Да | Нет |
| Исчезают | Нет (можно перевыполнить) | Заменяются новым |
| Награды | Фиксированные | Рандомные |

## Система эскорта (тип "Эскорт")

NPC-подопечный ведёт себя как наёмник-компаньон (система уже есть):
- Следует за игроком
- Имеет HP (50), может умереть
- Если умер — квест провален
- Не атакует, только идёт
- При достижении цели — квест выполнен, NPC исчезает

## Босс-рематч (тип "Босс-рематч")

- Только для уже побеждённых боссов
- Босс респавнится на своей карте с полным HP
- После убийства — не даёт артефакт повторно, только монеты/XP
- `defeatedBosses` не влияет — квестовый респавн отдельный

## Файлы

| Файл | Действие | Описание |
|------|----------|----------|
| `js/questgen.js` | Создать | Шаблоны, генерация, рандомизация, названия |
| `js/quests.js` | Изменить | Поддержка динамических квестов, серебряный цвет |
| `js/npc.js` | Изменить | Старейшина/Король/Страж предлагают рандомные квесты |
| `js/main.js` | Изменить | Обновление рандомных квестов, эскорт-NPC |
| `js/save.js` | Изменить | Сохранение активных рандомных квестов |

~500 строк, 1-2 сессии.
