# Система рандомных событий — Спецификация

## Цель

Добавить случайные события при заходе на боевые карты: сундуки, элитные враги, засады, торговец-путник, временные баффы, секретные порталы. Каждый заход на карту уникален.

## Архитектура

Один новый файл `js/events.js`. Вызывается из `loadMap()` в `main.js` после спавна врагов. Не модифицирует существующие карты — добавляет сущности поверх.

```
loadMap() → спавн врагов → generateEvents() → добавляет сундуки/элитных/NPC/порталы
```

Событие — объект `{ type, x, y, data }`. Генератор бросает кости по таблице вероятностей, выбирает 1-3 события, размещает на свободных тайлах карты.

## Таблица событий

| Событие | Шанс | Условие |
|---------|------|---------|
| Сундук (видимый) | 30% | Любая боевая карта |
| Элитный враг | 25% | Любая боевая |
| Засада бандитов | 15% | Лес, ущелье, замок |
| Торговец-путник | 15% | Любая карта |
| Скрытый сундук | авто | При спавне элитного-Вождя |
| Временный артефакт-бафф | 7% | Любая боевая |
| Секретный портал | 3% | Любая боевая |

Минимум 1 событие гарантировано. Максимум 3 за заход (не считая автоматических типа скрытого сундука от Вождя).

## Элитные враги

Базовый враг берётся из спавнов текущей карты. Три типа:

| Тип | Визуал | HP | ATK | Дроп |
|-----|--------|----|-----|------|
| Яростный | Красная пульсирующая аура | x1 | x2 | Обычный, x3 XP/монет |
| Громила | Размер x1.5 | x2 | x1 | Обычный, x3 XP/монет |
| Вождь | Корона над головой | x1.5 | x1.5 | Гарантированный (оружие/броня) + скрытый сундук рядом, x3 XP/монет |

При смерти — частица "ЭЛИТА!" золотым.

## Сундуки

### Видимые

Тайл: коричневый ящик с золотой застёжкой. Игрок подходит, нажимает E.

| Редкость | Шанс | Содержимое |
|----------|------|------------|
| Обычный | 50% | 10-30 монет + 1 зелье |
| Хороший | 35% | 30-80 монет + рандомное оружие текущего тира |
| Редкий | 15% | 80-200 монет + оружие следующего тира |

### Скрытые

Появляются на месте смерти Вождя. Всегда "Хороший" или "Редкий" (50/50).

## Временные баффы

Светящийся камень на земле. Подбирается при наступании. Длительность 30 секунд. Иконка-таймер в HUD. Одновременно только один бафф, новый заменяет старый.

| Бафф | Эффект | Цвет свечения |
|------|--------|---------------|
| Ярость | x2 ATK | Красный |
| Щит богов | Бессмертие | Золотой |
| Ветер | x2 скорость | Голубой |
| Вампиризм | +5 HP за убийство | Зелёный |

## Торговец-путник

Обычный NPC, зелёный плащ с капюшоном. Рандомное имя из: "Странник Морис", "Торговка Зара", "Бродяга Финн", "Путница Эльза".

### Уникальные товары

| Предмет | Цена | Эффект |
|---------|------|--------|
| Большое зелье x3 | 60$ | +60 HP, 3 штуки |
| Свиток телепорта | 100$ | Мгновенный возврат в деревню |
| Эликсир силы | 150$ | Перманентно +2 ATK |
| Эликсир жизни | 150$ | Перманентно +20 max HP |
| Камень воскрешения | 200$ | Авто-респавн при смерти (1 использование) |

### Товары следующего тира

| Карта | Продаёт из |
|-------|-----------|
| Лес | Ущелье (стальной меч, кольчуга) |
| Ущелье | Пещера (мифриловый меч, железная броня) |
| Пещера | Замок (рыцарское оружие) |
| Замок | Божественный тир (только эликсиры) |

Путник предлагает 3-4 рандомных товара из обоих списков.

## Засада

Условие: карты лес, ущелье, замок.

При срабатывании:
1. Текст "ЗАСАДА!" красным по центру экрана
2. 4 бандита (мечник, копейщик, лучник, топорщик) спавнятся вокруг игрока в радиусе 3-4 тайлов
3. За победу — x2 лут от каждого бандита

## Секретный портал

Фиолетовый мерцающий портал на случайном свободном тайле.

При входе генерируется комната 10x10:
- Стены по краям
- 3-5 врагов (тир текущей карты +1)
- 2 видимых сундука (всегда "Хороший" или "Редкий")
- Портал обратно на карту в ту же позицию

## Файлы

| Файл | Действие | Описание |
|------|----------|----------|
| `js/events.js` | Создать | Генератор событий, таблицы, спавн |
| `js/main.js` | Изменить | Вызов generateEvents в loadMap, баффы в updatePlayer, HUD таймер баффа, рендер сундуков/артефактов |
| `js/enemies.js` | Изменить | Флаги элитных: _eliteType, _eliteScale, рендер ауры/короны/размера |
| `js/sprites.js` | Изменить | drawChest, drawBuffStone, drawSecretPortal |
| `js/npc.js` | Изменить | Диалоги торговца-путника |
| `js/combat.js` | Изменить | x3 XP/монет для элитных |
